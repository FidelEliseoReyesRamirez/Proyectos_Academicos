# Proyectos Académicos Univalle

Sistema web desarrollado con Laravel, React, Inertia, Docker y PostgreSQL para la gestión de proyectos académicos, usuarios, roles, autenticación y seguimiento académico.

---

## 1. Requisitos previos

Antes de ejecutar el proyecto, cada integrante del equipo debe tener instalado lo siguiente.

### Obligatorio

- Git
- Docker Desktop
- Docker Compose
- PHP 8.3 o superior
- Composer
- Node.js 20 o superior
- npm
- Visual Studio Code o editor equivalente

### En Windows

Se recomienda tener:

- Windows 10 o superior
- WSL2 activado
- Docker Desktop configurado con WSL2
- Git Bash instalado

Verificar instalaciones:

```bash
git --version
docker --version
docker compose version
php -v
composer -V
node -v
npm -v
